# test

## 创建版本库

```git
git init     
git init <指定目录>
```

## 添加新文件或修改文件到暂存区

```    
git add .  
git add <指定文件>
git add <文件1> <文件2> <文件3> <文件...>
```

## 提交缓存区中的更改到本地仓库

```
git commit -m "提交注释"
```

## 克隆项目
```
git <仓库地址>
git <仓库地址> <指定目录>
```

## 查看仓库的当前状态

```
git status
git status -s        #获得简短的结果输出
```
